
# Lotto Number Generator

## Overview

This project is a simple web application that generates and displays random lottery numbers. It is built using modern web standards, including HTML, CSS, and JavaScript, and leverages Web Components for a modular and reusable UI.

## Design and Features

*   **Aesthetics:** The application will have a clean and modern design with a focus on usability. The lottery numbers will be displayed in a visually appealing way, resembling lottery balls.
*   **Interactivity:** Users can generate a new set of lottery numbers by clicking a button.
*   **Components:**
    *   **Lotto Ball:** A custom element (`<lotto-ball>`) will be used to display each number. This component will encapsulate its own styling and behavior.
*   **Layout:** The layout will be responsive and centered on the page.
